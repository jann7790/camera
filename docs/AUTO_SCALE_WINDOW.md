# 自動視窗縮放功能

## 問題描述

當相機設置為旋轉 90 度時，圖像尺寸從 1920×1200 變成 1200×1920（豎直方向），導致視窗高度超出螢幕範圍。

## 解決方案

添加了智能自動縮放功能，會自動檢測圖像大小並調整到適合螢幕的尺寸。

## 使用方法

### 方法 1: 配置文件（推薦）

編輯 `camera_config.yaml`:

```yaml
display:
  scale: auto  # 自動適應螢幕（預設）
  # scale: 0.5  # 或固定縮放 50%
  # scale: 1.0  # 或不縮放（原始大小）
```

### 方法 2: 命令行參數

```bash
# 自動適應螢幕（預設）
./run_camera.sh --scale auto

# 或固定縮放 50%
./run_camera.sh --scale 0.5

# 或不縮放
./run_camera.sh --scale 1.0

# 簡短參數
./run_camera.sh -s 0.5
```

### 結合旋轉使用

```bash
# 旋轉 90 度 + 自動縮放
./run_camera.sh -r 90 --scale auto

# 旋轉 90 度 + 翻轉 + 自動縮放
./run_camera.sh -r 90 --flip --scale auto
```

## 自動縮放邏輯

當 `scale: auto` 時：

1. 假設螢幕可用區域:
   - 最大寬度: 1900 像素
   - 最大高度: 1000 像素（留空間給任務欄、標題欄等）

2. 計算縮放比例:
   ```
   scale_width  = 1900 / image_width  (如果圖像寬度 > 1900)
   scale_height = 1000 / image_height (如果圖像高度 > 1000)
   scale = min(scale_width, scale_height)
   ```

3. 只在需要縮小時才縮放（scale < 1.0）

## 示例

### 原始圖像 (1920×1200)
- 不需要縮放，直接顯示原始大小

### 旋轉 90° 後 (1200×1920)
- 高度 1920 > 1000，需要縮放
- scale = 1000 / 1920 ≈ 0.52
- 縮放後尺寸: 624×1000 像素
- ✅ 完整顯示在螢幕內

### 旋轉 90° + 縮放 0.5 (固定)
- 縮放後尺寸: 600×960 像素
- ✅ 更小，留更多空間

## 調整螢幕尺寸假設

如果你的螢幕較小或較大，可以修改代碼中的預設值。

編輯 `flir_camera_preview.py` 第 755-756 行:

```python
# 調整這兩個值以匹配你的螢幕
max_display_height = 1000  # 改成你的螢幕高度減去 200-300
max_display_width = 1900   # 改成你的螢幕寬度減去 100
```

常見螢幕解析度建議:

| 螢幕解析度 | max_width | max_height |
|-----------|-----------|------------|
| 1920×1080 | 1850 | 900 |
| 2560×1440 | 2450 | 1250 |
| 3840×2160 | 3750 | 2000 |
| 1366×768  | 1300 | 600 |

## 注意事項

1. **縮放只影響顯示，不影響保存的圖像**
   - 按 's' 保存時，會保存原始尺寸的圖像
   - 縮放僅用於預覽視窗

2. **使用 INTER_LINEAR 插值**
   - 平衡速度和質量
   - 適合即時預覽

3. **只縮小不放大**
   - 當 scale = 'auto' 時，只會縮小過大的圖像
   - 不會放大小圖像（避免模糊）

4. **優先級**
   - 命令行參數 > 配置文件
   - 如果兩者都沒設置，預設為 'auto'

## 疑難排解

### 視窗仍然太大
- 降低 max_display_height 和 max_display_width 的值
- 或使用固定縮放: `--scale 0.4`

### 視窗太小
- 使用更大的固定縮放: `--scale 1.5`
- 或修改 max_display_height/width 為更大的值

### 圖像看起來模糊
- 使用 `--scale 1.0` 顯示原始大小
- 或使用較小的縮放值如 `--scale 0.8`

### 想看完整圖像但不縮放
- OpenCV 視窗支持滾動條（如果圖像超出視窗）
- 或使用 `--scale 1.0` 並手動調整視窗大小

## 完整示例

```bash
# 典型使用場景: 旋轉 90 度 + 自動縮放
./run_camera.sh -r 90 -s auto -e 5000 -g 0

# 高 DPI 螢幕，使用較小縮放
./run_camera.sh -r 90 -s 0.3

# 不旋轉，固定縮放 70%
./run_camera.sh -s 0.7

# 使用配置文件中的設置（auto）
./run_camera.sh -r 90
```
